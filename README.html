Workflow
Analysis Pipeline
01
Database Creation & Cloud Setup
Raw sales data is uploaded to AWS S3 as the central data lake. It is then ingested into AWS RDS ‚Äî a managed relational database ‚Äî where it's cleaned, normalized, and structured. Tables are type-enforced and missing values handled, laying a solid foundation for all downstream analysis.
02
Exploratory Data Analysis
Statistical summaries, distribution plots, time-series trends, and outlier detection. Identifies which vendors, categories, or periods are driving or dragging performance ‚Äî before jumping to conclusions.
03
Vendor Performance Analysis
Ranks vendors by revenue, units sold, and profit margins. Compares contribution share to total business. Highlights high-value vendors and surfaces those needing attention or improvement.
04
Power BI Dashboard
Interactive visuals built in Power BI for business stakeholders. Slicers by vendor, time, and category. KPI summary cards, trend lines, and comparative bar charts make insights accessible to non-technical users.
